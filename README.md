## 技术框架
可视化技术：three.js、d3.js、leaflet.js、canvas粒子系统

前端框架：vue.js、electron

数据处理：python

## 代码运行方法

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
